1. Panoramica del Progetto

Cos'è Dashboard ARIMA?

Dashboard ARIMA è un'applicazione web completa per l'analisi automatica di serie temporali con modelli ARIMA/SARIMA. Costruita con Flask seguendo le best practices architetturali.

Funzionalità Principali

Upload File

CSV/XLSX con colonne 'data' e 'y'

Visualizzazione

Grafici interattivi della serie

Trasformazioni

Smoothing, logaritmo, differenziazione

Analisi ACF/PACF

Con suggerimenti automatici modello

Split Configurabile

Training/Test personalizzabile

Selezione Automatica

Miglior modello basato su R²/MAPE

Output Completo

Coefficienti, p-values, CI

Export

PDF, Excel, JSON, Paper HTML

5. app.py - Factory Pattern

Application Factory per istanze configurabili

backend/app.py 
def create_app():
    app = Flask(__name__, 
                template_folder='../frontend/templates',
                static_folder='../frontend/static')
    app.config.from_object(Config)
    
    # Inizializza Flask-Login
    login_manager.init_app(app)
    
    # Crea cartelle necessarie
    os.makedirs(Config.UPLOAD_FOLDER, exist_ok=True)
    os.makedirs(Config.EXPORT_FOLDER, exist_ok=True)
    
    init_db(app)
    app.register_blueprint(api, url_prefix='/api')
    
    # Route principale dashboard
    @app.route('/')
    def index():
        return render_template('dashboard.html')
    
    # Route per il paper HTML
    @app.route('/paper')
    def paper():
        return render_template('paper.html')
    
    return app
✓ Configurazioni diverse ✓ Test facili ✓ No import circolari

9. ai_service.py

490 righe • Analisi ACF/PACF

Algoritmo di Suggerimento

1Calcola soglia: 2.0 / √n
2Identifica lag significativi ACF/PACF
3Valuta pattern: lento vs cut-off
4Stima ordini p, d, q
5Rileva stagionalità (lag ≥ 4)
6Suggerisci trasformazioni (test ADF)

SQLite (Attuale)
  • Zero configurazione
  • Database come singolo file
  • Perfetto per sviluppo
  • Backup = copia file
Limitazioni
  • Un solo writer alla volta
  • Lock sul file
  • Nessuna condivisione tra server
  • Non adatto per alta concorrenza

Glossario - Architettura

Termini architetturali

MonoliteArchitettura con singolo deploy e processo. Non significa disordinato.
Application FactoryPattern Flask: create_app() per istanze configurabili
BlueprintModulo Flask per organizzare route in gruppi separati
